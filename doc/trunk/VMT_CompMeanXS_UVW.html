VMT_CompMeanXS_UVW

PURPOSE

Computes the mean cross section velocity components (U,V,W)

SYNOPSIS

function [A,V] = VMT_CompMeanXS_UVW(z,A,V)

DESCRIPTION 

 Computes the mean cross section velocity components (U,V,W)
 from individual transects that have been previously mapped to a common
 grid and averaged.
