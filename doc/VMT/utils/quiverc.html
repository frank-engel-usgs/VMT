quiverc

PURPOSE

Modified version of Quiver to plots velocity vectors as arrows

SYNOPSIS

function hh = quiverc(varargin)

DESCRIPTION 

 Modified version of Quiver to plots velocity vectors as arrows 
 with components (u,v) at the points (x,y) using the current colormap
